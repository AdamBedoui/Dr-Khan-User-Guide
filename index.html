How to install the website

I have provided a single file installer using " All-in-One WP Migration " plugin, which allows importing, exporting and backing up websites an easy task. The file extension is .wpress.

The installer includes all the code sources, assets, configurations of the website, such as :

  • Site settings
  • All the templates
  • The posts' and pages' contents
  • The media files, such as images, videos, document files...
  • The fonts and colors used in the website.
  • The necessary plugins with their

I recommend starting fresh with a new WordPress instance.

The link for the file installer will be provided separately.

After downloading the file, go to your WordPress dashboard, hover over Plugins, then Add New

Then in the search input, search for All-in-One WP Migration find the one developed by ServMask and then click on Install now

Then click on Activate Plugin

After that, hover on All-in One WP Migration then Import

Make sure that the Maximum upload file size of your WordPress is enough to import the installation file.

Otherwise, I recommend checking this article to fix that

How to Increase the Maximum File Upload Size in WordPress

Then, click on Import From FILE

And Open the Setup file

Wait a few seconds then click on Proceed

Wait untill the import finishes, then cick on Finish

Now refresh the page, and login to dashbord using the login information that will be provided separetely

That's it ! You successfully installed the website !

Now you can open the front site and check that everything is working well.
